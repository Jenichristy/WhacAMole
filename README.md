# Whac-A-Mole Game 🎮

A simple Whac-A-Mole mini project built using **Java Swing**.  
Click the mole to gain points, avoid the plant to prevent game over!

---

## Features
- 3x3 grid gameplay
- Random mole and plant appearance
- Score tracking
- Game over condition when clicking plant

---

## Requirements
- Java JDK 17 (or higher)
- Git installed (for cloning)

---
